Comando: git clone

El comando git clone se utiliza para copiar un repositorio remoto a tu ordenador local.

Sintaxis

git clone <url-del-repositorio>

¿Qué hace?

  • Descarga todos los archivos del repositorio
  • Crea una carpeta con el nombre del proyecto
  • Configura automáticamente la conexión con el repositorio remoto

Ejemplo

git clone https://github.com/usuario/proyecto.git

Cuándo usarlo

Usa este comando cuando quieras empezar a trabajar con un repositorio que ya existe en plataformas como GitHub.
